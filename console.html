Dashboard

This console is to help customers choose parameters like Usable capacity, Disk Type, RAID Type from the front end and also check out the typical values of IOPS, Throughput and total number of data disks, parity disks, RPM, etc.


Guide to use the console:


  1. Use the Select tab to enter the input details.
  2. Use the BOM tab to see the Billing of Materials section
  3. Use the Pricing tab to get a cost estimate of the order.
  4. Use the inventory to check out features of all available products.
